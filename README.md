# The Loom: dashboard

loom es una aplicación web que acompaña a los estudiantes en su cursada

## Setup

```shell
  cp .env.example .env
  nano .env
  docker-compose build
```

## Running the app

```shell
  make start
```

## Stopping the app

```shell
  make stop
```
